Overview

Run for Migrant Heroes 2021 is a virtual fundraiser run initiated by nbwu.sg team. All proceeds collected from this event will be used to provide welfare for migrant workers. We are working with Transient Workers Count Too (TWC2) as the bridge between the team and the migrant worker community.

How will the Migrant Worker Community benefit?

The money collected will be used to purchase essentials such as soap, shampoo, hand sanitisers. The essentials will be packaged into a goodie bag and given to the migrant workers, including handwritten postcards by our runners. The postcard is to show appreciation for the migrant workers and to let them know that they are supported.

What is a virtual run?

A Virtual Run is exactly the same as a marathon run just that it can be done at any location (outdoors or indoors on the treadmill) and at any time.
